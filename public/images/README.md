# Images Directory

Place all images in this directory. The website expects the following images:

## Required Site Images

| File | Description | Recommended Size |
|------|-------------|-----------------|
| `hero.jpg` | Homepage hero image | 1920x1280 (3:2 ratio) |
| `about.jpg` | About preview on homepage | 1200x1500 (4:5 ratio) |
| `about-hero.jpg` | About page hero image | 1200x1500 (4:5 ratio) |
| `og-image.jpg` | Social sharing image | 1200x630 |
| `favicon.ico` | Browser tab icon | 32x32 (also place in /public/) |

## Image Guidelines

1. **Format**: Use `.jpg` or `.webp` for photos
2. **Optimization**: Compress images before uploading (tinypng.com)
3. **Naming**: Use lowercase, hyphens instead of spaces
4. **Alt text**: Already defined in the code, but describe images meaningfully

## Temporary Placeholders

Until real photos are added, the site will show a warm stone gray placeholder where images would appear. This is intentional - the site remains fully functional.

To use placeholder images during development, you can use services like:
- unsplash.com (high-quality free photos)
- placeholder.com (gray boxes with dimensions)
